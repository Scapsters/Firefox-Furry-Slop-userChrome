The client for this server is a css file that must be created manually.

1. Go to `about:support` on your browser (This is for Firefox)
2. Under Application Basics, find Profile Folder
3. In that directory, create a folder named `chrome`
4. In the chrome folder add `userChrome.css` from this repository
5. Run this server with `npm run start`
6. Restart Firefox fully (Whatever that means)
7. :3
